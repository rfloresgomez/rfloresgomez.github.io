Ámbito de las variables JS:

Si declaramos una variable con var, esa variable es global, aunque la declaremos dentro de un bloque. Para que una variable no sea global y sea de bloque, debemos crearla con let dentro de ese bloque.

Hoisting en las variables:

Consiste en que las variables que creamos en nuestro código, son declaradas al principio del bloque aunque las hayamos creado en otra parte del código, lo que significa que si queremos acceder a ellas antes de inicializarlas, nos devolverá undefined.

Hoisting en las funciones:
Las variables que creemos dentro de una función, también hacen hoisting.

Palabras reservadas de la declaración:

  • var
  • let
  • const

Declaración/invocación de funciones

Podemos declarar una función de distintas formas:
  • Poniendo directamente: 
    function nombreFuncion(parametros){}
  • Igualando una variable a una función anónima: 
    let miFuncion = function(parametros){}
